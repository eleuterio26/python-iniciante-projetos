# Projetos em Python (Iniciante)

Este repositório contém projetos desenvolvidos para prática de lógica de programação em Python.

## Projetos

### 📌 Calculadora de IMC
Arquivo: `imc.py`  
- Cálculo de índice de massa corporal  
- Uso de entrada de dados e condicionais  

### 📌 Cadastro de Pacientes (Simples)
Arquivo: `cadastro_pacientes.py`  
- Armazenamento de dados em memória  
- Uso de listas e dicionários  

### 📌 Sistema de Pacientes (Completo)
Arquivo: `sistema_pacientes.py`  
- Menu interativo  
- Cadastro, listagem e busca  
- Salvamento de dados em arquivo (.txt)  

## Tecnologias
- Python
